# Swiggy_data_analysis_dashboad_Project
A data analysis project based on Swiggy restaurant dataset where raw data was cleaned and analyzed to build an interactive dashboard and extract insights about restaurants, ratings, cuisines, and pricing trends using Microsoft Excel.

## Project Overview
This project focuses on analyzing a Swiggy restaurant dataset to uncover insights about restaurant ratings, cuisines, pricing, and restaurant distribution.

## Data Cleaning Process
* The dataset was cleaned using the following steps:
* Removed duplicate records
* Handled missing values
* Standardized column names
* Converted numeric fields
* Organized categorical variables

📊 Dashboard Features

The dashboard provides multiple insights through visualizations:
⭐ Restaurant Rating Analysis
🍜 Cuisine Distribution
💰 Cost for Two Analysis
🏙 City-wise Restaurant Distribution
📈 Top Rated Restaurants
Interactive filters allow users to explore the data dynamically.

## Key Insights

* Some cuisines dominate the restaurant listings on Swiggy.
* Highly rated restaurants tend to have more customer votes.
* Pricing varies significantly across restaurants.
* Certain cities have a higher density of restaurants.

The analysis of the Swiggy restaurant dataset provided valuable insights into restaurant trends, cuisine popularity, pricing patterns, and customer preferences.
